
# LPG-SMARTGUARD: IoT-Based Smart LPG Leakage Detection and Auto-Refill System

This project is an IoT-based smart safety system designed to detect LPG gas leakage in real time and provide immediate alerts to users. The system uses gas sensors and a microcontroller to continuously monitor LPG concentration levels. When leakage is detected, the system activates a buzzer alert and sends notifications through wireless communication to ensure quick response and improved safety.

In addition to leakage detection, the project also includes an automatic LPG refill feature that can initiate refill requests when the gas level becomes low. The system enhances home and industrial safety, reduces accident risks, and improves user convenience through automation and real-time monitoring.

## Features

* Real-time LPG leakage detection
* Instant buzzer and alert notification
* IoT-based wireless monitoring
* Automatic LPG refill request system
* Low-cost and energy-efficient design
* Easy integration with smart home systems

## Technologies Used

* Arduino / ESP32
* LPG Gas Sensor (MQ-2 / MQ-6)
* Embedded C Programming
* IoT Communication
* PCB Design using KiCad
* Sensor Interfacing
